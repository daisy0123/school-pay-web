#### 基于nodejs的校园二手交易平台

##### 环境：
1. nodejs version：6.x
2. npm version： 3.x

- 第一步 拉代码
```
git clone https://gitlab.com/school-pay/school-pay.git
```
- 第二步 安装依赖
```
npm install
```
如果有sass-loader安装不成功，可使用[淘宝镜像](https://npm.taobao.org/)进行安装
```
npm install -g --registry=https://registry.npm.taobao.org
```
- 第三步 启动项目
```
npm run dev
```
